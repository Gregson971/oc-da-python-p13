Bienvenue dans la documentation d'OC-lettings !
===================================================

Orange County Lettings est une start-up dans le secteur de la location de biens immobiliers. 
La start-up est en pleine phase d’expansion aux États-Unis.
Sur le site, vous pouvez consulter plusieurs lieux de location ainsi que les profils
des utilisateurs.

Technologies et langages de programmation
-----------------------------------------

Cette application est programmée en Python et Django.

Les données sont stockées dans une base de données SQLite3.

Les différentes plateformes utilisées sont :

   - GitHub, pour sauvegarder le code et chacune de ses versions,
   - GitHub Actions, pour automatiser les tests et le déploiement du code,
   - Sentry, pour surveiller les performances du site,
   - Docker, pour une conteneurisation du code,
   - Azure, pour le déploiement du site.
